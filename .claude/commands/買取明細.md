# 買取明細PDF生成

ユーザーから以下の情報を受け取り、`generate_receipt.py` を呼び出してPDFを生成する。

## 入力フォーマット（ユーザーが送る情報）

- **日付**（例: 2026年3月5日）
- **宛名**（会社名 or 氏名）
- **住所**（〒郵便番号 住所）
- **受付ID**（備考欄に記載）
- **商品明細**（品名、数量、単価）
- **支払予定日**（省略時は空欄→「別途通知」）

## ルール

- 法人（商店・会社など）→ `honorific="御中"`（デフォルト）
- 個人（氏名＋様）→ `honorific="様"`
- `receipt_number` は指定しない（自動採番）
- `output_dir` は `/Users/natsuki/Documents/1000_買取明細`
- 支払方法は常に `銀行振込`

## 実行

ユーザーから買取明細データを受け取ったら、Bashで以下を実行する。

```python
from generate_receipt import generate_receipt

generate_receipt(
    date="...",
    recipient_name="...",
    recipient_address="〒xxx-xxxx　...",
    subject="...様/御中の...買取に関する支払通知",
    items=[
        {"name": "...", "qty": N, "unit_price": N},
        ...
    ],
    payment_date="...",   # 省略時は ""
    payment_method="銀行振込",
    note="受付ID：...",
    output_dir="/Users/natsuki/Documents/1000_買取明細",
    honorific="御中",  # 個人の場合は "様"
)
```

生成後、ファイル名・支払金額サマリー（小計・消費税・お支払金額）を返す。
